Install vs. repair

New door, hung right
the first time.

Installing a pre-hung door is different from repairing one — you're setting the frame in the rough opening, shimming it plumb and level, and making sure the reveal gap is consistent before nailing it off. In a 1970s or 1980s Longwood home, that rough opening is rarely perfectly square. We account for that.

Exterior door installation includes the threshold, weatherstripping, and door sweeps. We don't leave gaps that let Florida humidity — or bugs — in. If the sill needs to be shimmed or the threshold needs to be cut to the subfloor profile, that's part of the job.

We do pre-hung door replacement like-for-like on interior and exterior doors. New construction rough-ins or structural door opening modifications (widening a doorway, adding a door where there wasn't one) require a permit and are outside our scope.

What's included

Every job, every time

Pre-hung door install

Frame set plumb and level in the existing rough opening. Shimmed at hinges, latch, and top. Consistent reveal gap all the way around. We don't skip the last 10% that makes a door feel right.

Exterior weatherproofing

Common questions

Good questions, straight answers

Do you supply the door or do I?

Either way. If you've already bought the door, we install it. If you haven't, tell us the opening size and application (interior/exterior, swing direction) and we can source and bring the door. We'll add the door cost to the quote.

Does door replacement require a permit in Longwood?

Like-for-like replacement — same size opening, same door type — does not require a permit in Seminole County. Modifying the rough opening size or adding a new door where there wasn't one does. We won't do unpermitted work that requires a permit.

My door opening isn't square — can you still install it?

Yes, that's normal in older Longwood homes. Out-of-square openings require more shimming and sometimes trimming the door frame or adding to it. We assess that on-site before quoting. If the out-of-square is significant enough to affect the door's function, we'll tell you before we start.

What's the difference between door repair and door installation?

Repair is fixing what's there — hinge adjustment, strike plate relocation, planing a sticking door, replacing a broken pane. Installation is replacing the entire door unit in the opening. We do both — repair is at /services/door-repair, this page covers full replacements.
